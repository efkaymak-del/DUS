## 🚀 Özellikler

- **📝 Tam Deneme Sınavı** — 120 soru (40 Temel + 80 Klinik), 150 dakika zamanlayıcılı, gerçek DUS formatında
- **📚 Hap Bilgiler** — 10+ dal, öncelik ve çıkma oranı etiketli akordeon kartlar
- **🃏 Flashcards** — 20 kart, 3D flip animasyonu, biliyorum/tekrar et takibi
- **🧮 Puan Hesaplayıcı** — Net = Doğru − (Yanlış ÷ 4), uzmanlık dalı yerleşme tahmini
- **📅 Sınav Takvimi** — ÖSYM resmi takviminden güncel tarihler ve geri sayım
- **🔗 Kaynaklar** — ÖSYM, TDB, YÖK, çıkmış sorular, hazırlık kursları linkleri
- **📱 Responsive** — Mobil, tablet, masaüstü uyumlu
- **🌙 Tamamen istemci taraflı** — Sunucu gerektirmez, GitHub Pages'da barındırılabilir
